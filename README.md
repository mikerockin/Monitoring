# Monitoring
Scripts for Prometheus Monitoring System

### Includes  scripts:

    1. Install Prometheus 
    2. Install Grafana
    3. Install Prometheus Node Exporter
    4. Install Prometheus Mysqld Exporter
    5. Install Prometheus Alert Manager
